# Implement-an-AVL-Tree
implement a simple linked list based binary tree. 


Instructions
Implement a template AVL tree with the following functionality:

Please implement the following functionality
push function (with self-balancing)
remove  (with self-balancing)
clear
copy
Use an enum to set the traversal of the tree
PREORDER
INORDER
POSTORDER
Print the tree in the in the various traversal order
There should be one print function that will print the tree according to the set traversal order.
Execute a function passed to your traversal functions
Please draw your tree and all of the traversals to verify your tree is implemented correctly
Dont forget the Big 3
